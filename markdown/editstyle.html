# 第一章节

Markdown 是一种轻量级的标记语言，用于格式化文本。以下是常用的 Markdown 语法总结：

# 标题

使用 # 来创建标题。# 的数量决定了标题的级别，数量越大，标题字体越小，最高支持 6 级

# 一级标题 <h1>

## 二级标题 <h2>

### 三级标题 <h3>

#### 四级标题 <h4>

##### 五级标题 <h5>

###### 六级标题 <h6>

# 代码段

使用 ``` 代码后缀 来创建代码块。后缀 html 代表语法格式

    ``` html
        <h1>这是代码内容<h1>
    ```

# 列表

使用 - 代码 来创建 1 2 3 4 分别用空格对齐

- 列表 1
  - 列表 1.1
    - 列表 1.1.1
  - 列表 1.2
- 列表 2
- 列表 3

# 表格

使用 | 代码 来创建

| 格式 | 输出 | 描述         |
| ---- | ---- | ------------ |
| YY   | 18   | 两位数的年份 |

# 提示框

绿 tip 黄 warning 红 danger 灰 details

    <!-- 绿 -->
    <div class="custom-block tip">
        <p class="custom-block-title">TIP</p>
        <p>默认格式为 </p>
    </div>
    <!-- 黄 -->
    <div class="custom-block warning">
        <p class="custom-block-title">WARNING</p>
        <p>warning 提示框</p>
    </div>
    <!-- 红 -->
    <div class="custom-block danger">
        <p class="custom-block-title">DANGER</p>
        <p>warning 提示框</p>
    </div>
    <!-- 灰 -->
    <div class="custom-block details">
        <p class="custom-block-title">DETAILS</p>
        <p>details 提示框</p>
    </div>

TIP

tip 提示框

WARNING

warning 提示框

DANGER

danger 提示框

DETAILS

details 提示框

# code

使用 <code> 标签 来包括：内容

    <code>这是代码内容<code>

# <Array>

使用 Promise<Array> 类似代码，页面报错或直接空白的情况

    ``` html
    <!-- &lt;内容&gt; -->
        Promise&lt;Array&gt;

    <!-- &lsqb;内容&rsqb; -->
        Promise&lsqb;Array&rsqb;

    ```

这些扩展语法在一些支持扩展功能的 Markdown 渲染器中非常有用，特别是用于更复杂的文档或展示效果。
