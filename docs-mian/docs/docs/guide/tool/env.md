# 宿主信息

判断宿主环境信息的函数集合

## isDesktop

参数：无
返回值：{Boolean}
说明：是否为桌面端

## isMobile

参数：无
返回值：{Boolean}
说明：是否为手机端

## isTablet

参数：无
返回值：{Boolean}
说明：是否为平板端

参数：无
返回值：{Boolean}

## isAndroid
