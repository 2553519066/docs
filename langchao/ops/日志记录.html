操作日志查询接口文档

接口基本信息

项目内容
接口地址/api/platform/log
请求方式GET
接口描述查询系统操作日志信息

请求参数

参数名类型是否必传说明
skipCountnumber是跳过条数
maxResultCountnumber是每页最大条数
filterstring否区域部门名称
userNamestring否用户名
roleNamestring否角色名
ykzUidstring否愉快政 UID
actionModulestring否操作模块名称
actionContentstring否操作内容
expandInformationstring否拓展信息
actionStatusboolean否操作状态
userAgentstring否User Agent
ipstring否操作 Ip
startTimestring是开始时间（需与 endTime 一起传入，格式建议为 YYYY-MM-DD HH:mm:ss）
endTimestring是结束时间（需与 startTime 一起传入，格式建议为 YYYY-MM-DD HH:mm:ss）

返回数据结构

字段名类型说明
RegionIdGuid区域 Id
RegionNamestring区域名称
BigDepartmentIdGuid大部门 Id
BigDepartmentNamestring大部门名称
DepartmentIdGuid部门 id
DepartmentNamestring部门名称
UserIdGuid用户 id
UserNamestring用户名
RolesList<string>角色信息（businessRole 字段，使用，分割）
YkzUidstring愉快政 UID
ActionModulestring操作模块
ActionContentstring操作内容
ActionTypestring操作类型
ExpandInformationstring拓展信息
ActionTimeDateTime操作时间（格式为 YYYY-MM-DD HH:mm:ss）
ActionStatusboolean状态
OperatingEquipmentstring操作设备
UserAgentstringUA
IpstringIP 地址
