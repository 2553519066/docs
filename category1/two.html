# 分类1-2

This site is built with VitePress (opens new window) and depends on @vue/theme (opens new window). Site content is written in Markdown format located in src. For simple edits, you can directly edit the file on GitHub and generate a Pull Request.

For local development, pnpm (opens new window) is preferred as package manager:

pnpm i
pnpm run dev

This project requires Node.js to be v18 or higher. And it is recommended to enable corepack:

corepack enable
